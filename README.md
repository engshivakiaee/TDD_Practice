# TDD_Desk_Booker

Test Driven Development Practice

## Description
Implementing a desk booker project with TDD

## Getting Started

### Running the program

Using Visual Studio F5, or if you are using NET cli, you can run the program by typing the following command in the terminal:

```
dotnet run
```

### Running the tests

```
dotnet test
```
